# Installation Guides

Purpose:

- Choose the right installation path for your tooling.

Scope:

- Use when setting up Router in a new project.

Guidelines:

- Start with manual setup for full control.
- Use tool-specific guides for best defaults.
- Align Router CLI usage with your build tool.

Examples:

```text
pnpm add @tanstack/react-router
```

```text
pnpm dlx @tanstack/router-cli@latest init
```
