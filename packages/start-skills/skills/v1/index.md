# Start Skill Index

Purpose:

- Help pick the right Start skill quickly.

Scope:

- Applies to any Start adapter or deployment target.
- Routes to focused skills for setup, server functions, and deployment.

How to pick a skill:

Core setup and structure:

- Bootstrapping a Start app -> `@skills/start/start-setup`
- Understanding app entry points and layout -> `@skills/start/file-structure`
- Wiring Start with TanStack Router -> `@skills/start/router-integration`

Runtime and server boundaries:

- Choosing and configuring adapters -> `@skills/start/adapters`
- Building typed server functions -> `@skills/start/server-functions`
- Central request/response middleware -> `@skills/start/middleware`
- Streaming SSR and suspense flows -> `@skills/start/streaming-ssr`

Deployment:

- Environment variables and secrets -> `@skills/start/environment-config`
- Deploying to a hosting target -> `@skills/start/deployment`

Next:

- After picking a skill, follow its resource links and guidance.
